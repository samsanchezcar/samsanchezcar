## Rules

1. Badge background: always `#1a1a2e`
2. Stats/graph background: always `#0d0d1a`
3. Primary accent: `#A78BFA` (logo colors, titles)
4. Secondary accent: `#C4B5FD` (desc, secondary text)
5. Inline badges: `style=flat-square`
6. Section badges: `style=for-the-badge`
7. skillicons: always `theme=dark`
8. Arduino → Tools section, never Languages
9. Divider gif: after main header + before footer
10. Typing SVG font: always `Fira+Code`
11. `demolab.com` for headers/footers (supports multiline) · `herokuapp.com` for looping subtitles
